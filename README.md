# Enterprise Operations Hub

<p align="center">A powerful dashboard for streamlined business operations and data management.</p>

## Introduction

Enterprise Operations Hub is a comprehensive ASP.NET Core MVC dashboard template built for enterprise-level operations management. It provides a modern, responsive interface for monitoring business metrics, managing workflows, and analyzing data.

## Features

- Responsive dashboard with analytics
- Multiple layout options (with/without menu, navbar, container)
- Authentication pages (Login, Register, Forgot Password)
- Error and maintenance pages
- User interface components
- Form elements and layouts
- Data tables
